## 项目1: 「记忆碎片」- 互动叙事可视化

**概念**: 用户输入关键词/照片，AI生成"记忆碎片"式的动态叙事网页，每个碎片代表一段回忆，最终拼成完整故事。

### 评分详情

| 维度 | 评分 | 说明 |
|------|------|------|
| 创意性 | ⭐⭐⭐⭐ | 叙事与碎片化视觉结合新颖 |
| 视觉效果 | ⭐⭐⭐⭐⭐ | 玻璃态、粒子效果、3D变换 |
| 技术难度 | ⭐⭐⭐ | React + GSAP + AI API |
| 叙事性 | ⭐⭐⭐⭐⭐ | 内核就是叙事，天然优势 |
| 可展示性 | ⭐⭐⭐⭐ | 现场输入即生成，互动性强 |
| 团队友好 | ⭐⭐⭐⭐ | 1-2人可完成核心 |

**推荐指数**: ⭐⭐⭐⭐⭐

### 技术栈建议
- Frontend: React/Vue + GSAP/TweenMax
- AI: GPT-4 API / Stable Diffusion
- 3D: Three.js (可选)

**概念**: 用户选择情绪标签，AI生成匹配的音乐+实时动态粒子可视化，颜色随音乐律动。

| 维度 | 评分 | 说明 |

| 技术难度 | ⭐⭐⭐⭐ | Web Audio API + Canvas/WebGL |

**推荐指数**: ⭐⭐⭐⭐
