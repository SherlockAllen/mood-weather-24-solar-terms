— 风格系统 —

1. 视觉风格

采用 UI/UX-Pro-Max 浅色水墨风，以宣纸质感为底，浓墨为字，营造清雅脱俗的传统意境。

元素色值说明
背景色#f8f6f2宣纸质感，温暖柔和
主文字色#2a2a2a浓墨色，对比度 11.5:1
次要文字色#6a6a6a淡墨，用于诗句与描述
辅助色#8a8a86节气名称、装饰元素
强调色#d5d0ca远山、云雾层叠

字体选择：

  • 节气名称: Songti SC, SimSun — 宋体，传统感
  • 诗句: Kaiti SC — 楷体，书法感
  • 描述文字: Noto Serif SC — 清晰易读

— 动画设计 —

3. 四层水墨动效

层级元素实现动画参数
1远山剪影SVG <path>静态 + 半透明层叠
2墨迹晕染SVG <ellipse> + feGaussianBlurcx 位移 10s, opacity 呼吸 10s
3飘落墨点SVG <circle>cy 下落 12-18s, opacity 淡入淡出
4底部飘雾SVG <path> + animated 路径形变 8s, 循环流动

页面切换动画

  • 触发：点击箭头 / 键盘 / 圆点 / 触摸滑动
  • 效果：当前页淡出 + 新页淡入，配合 translateX 轻微位移
  • 时长：350ms
  • 缓动：cubic-bezier(0.4, 0, 0.2, 1)
  • 性能：仅 transform + opacity，GPU 加速

Reduced Motion 支持

检测 prefers-reduced-motion: reduce，关闭连续动画，保留页面切换但缩短至 150ms。

触摸优化

  • 箭头点击区域：44px × 44px（符合 Apple HIG）
  • 圆点点击区域：24px × 24px（热区扩大）
  • 元素间距：≥8px，防止误触
  • 滑动阈值：水平位移 ≥50px 触发切换
— 技术方案 —

5. 技术栈

HTML5 Tailwind CSS v4 SVG Vanilla JS

为什么选 Tailwind CSS v4

  • 零运行时：编译为纯 CSS，无 JS 运行时开销，首屏加载快
  • JIT 引擎：只生成用到的 class，CSS 体积极小（预计 < 10KB gzipped）
  • CSS 自定义属性：v4 原生支持 @theme，与水墨风颜色系统无缝结合
  • 响应式内置：sm/md/lg/xl 断点与设计方案对齐

自定义主题配置

@theme { --color-paper: #f8f6f2; --color-ink-dark: #2a2a2a; --color-ink-mid: #6a6a6a; --color-ink-light: #8a8a86; --color-mountain: #d5d0ca; --font-song: 'Songti SC', 'SimSun', serif; --font-kai: 'Kaiti SC', serif; --font-serif: 'Noto Serif SC', serif; }

分工：Tailwind 负责布局、间距、响应式、颜色、圆角、阴影；自定义 CSS 负责 SVG 动画关键帧、复杂渐变、特殊交互状态。

性能策略

  • 零图片：纯 CSS + SVG，无位图请求
  • 系统字体：Songti SC / Kaiti SC / Noto Serif SC，无需加载外部字体
  • GPU 加速：CSS @keyframes + transform
  • 事件节流：resize 和 touch 事件节流处理
  • CSS 体积：Tailwind v4 JIT + 自定义主题，最终 CSS < 15KB gzipped

无障碍支持

  • 对比度 11.5:1，远超 WCAG 2.1 AA 标准
  • 完整键盘导航，可见 focus 状态
  • ARIA 标签：role="region"、aria-label、aria-selected
  • 语义化标题：h1（sr-only）+ h2（节气名）
