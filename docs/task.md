# 情绪天气 v4 - 震撼视觉效果优化计划

> **目标**: 让每个节气拥有独特的情感共鸣和震撼时刻
> **创建时间**: 2026-05-18
> **优先级**: P0 > P1 > P2

---

## 问题诊断

| 维度 | 当前状态 | 问题 |
|------|----------|------|
| 情感传达 | 24节气共用6种特效 | 缺"情" |
| 动态层次 | 装饰多为静态 | 缺"动" |
| 震撼时刻 | 平滑但平淡 | 缺"爆点" |
| 视觉深度 | 4层扁平 | 缺"纵深" |
| 互动可能 | 纯展示无交互 | 缺"交互" |

---

## 优化方向（按优先级）

### P0 - Hackathon震撼时刻（必须实现）

- [x] **夏至"无影"效果** - 影子消失的瞬间
- [x] **冬至"月光雪景"** - 极致唯美的银蓝世界
- [x] **清明"烟雨蒙蒙"** - 情感共鸣的细雨场景

- [x] **季节过渡动画** - 冬→春/夏→秋戏剧化

### P2 - 加分项

- [x] **互动彩蛋设计** - 停留/长按/摇晃触发
- [ ] **音效配合**（可选）

## 详细实现计划

### 任务1: 夏至"无影"效果实现

**文件**: `src/carousel.js`, `src/animations.css`

**实现要点**:
```javascript
// 夏至检测
if (term.id === 10) { // 夏至
  // 1. 太阳光晕突然变强
  // 2. 所有阴影opacity→0
  // 3. 画面变"无影"
}
```

**CSS动画**:
```css
/* 夏至无影模式 */
.solstice-no-shadow .mountain-front,
.solstice-no-shadow .ink-blur-1,
.solstice-no-shadow .ink-blur-2 {
  opacity: 0;
}
```

### 任务6: 季节过渡动画

**文件**: `src/carousel.js`, `src/animations.css`

**冬→春 过渡**:
1. 雪花停止飘落
2. 所有白色渐变为暖黄
3. 地面出现绿色
4. 花朵绽放动画

**夏→秋 过渡**:
1. 烈日变为柔和
2. 绿叶变黄
3. 落叶飘落

---

### 任务7: 互动彩蛋

**文件**: `src/interactions.js`, `src/carousel.js`

| 彩蛋 | 触发方式 | 效果 |
|------|----------|------|
