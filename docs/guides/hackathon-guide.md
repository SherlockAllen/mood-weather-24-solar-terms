## 为什么参加 Hackathon？

1. **积累项目经验** - 在有限时间内完成完整项目，提升实战能力
2. **赚取美元** - 奖金从 $500 到 $50,000 不等
3. **拓展人脉** - 认识项目负责人、开发者、投资人
4. **开启远程兼职** - 优秀项目可能被赞助商看重，获得长期合作机会
5. **Vibe Coding 加持** - AI 辅助开发让个人参赛成为可能

### MeDo 是什么？

一个**自然语言生成应用**的平台。你用自然语言描述需求，AI 帮你生成完整应用：

- 前端 + 后端 + 数据库
- 一键部署到公网 URL
- 支持插件和 API 集成
- 支持多轮对话迭代修改

### 你的技能如何匹配？

| 你的技能 | 在 MeDo 中的用处 |
|----------|------------------|
| Vue | 理解生成的前端代码逻辑 |
| Python | 如果需要自定义后端逻辑 |
| Docker | 部署时理解容器概念 |
| ELK | 可视化应用数据分析 |

## Vibe Coding 参赛策略

### 什么是 Vibe Coding？

用 AI（Claude/GPT/Gemini）辅助编程，你负责：
- 需求分析
- 架构决策
- 代码整合
- UI/UX 把控
- 最终测试

AI 负责：
- 代码生成
- Boilerplate 编写
- 文档撰写
- Bug 修复

### 为什么 Vibe Coding 适合 Hackathon？

| 优势 | 说明 |
|------|------|
| **时间效率** | Hackathon 通常只有 24-48 小时 |
| **快速 MVP** | AI 能快速生成可用的原型 |
| **Demo 导向** | Hackathon 评分看 Demo，不看代码质量 |
| **降低门槛** | 一个人也能完成完整项目 |

### 推荐工具组合

```
提示词工程 → Claude / GPT-4
前端 → Vue + Tailwind CSS
后端 → Python FastAPI / Node.js
部署 → Vercel / Railway / Docker
数据库 → Supabase / PlanetScale
```

### 快速 MVP 提示词模板

```
我要做一个 [项目类型]，目标用户是 [用户群体]。
核心功能：
1. [功能1]
2. [功能2]
3. [功能3]
要求：
- 界面简洁现代
- 响应式设计
- 至少包含一个 AI 交互功能
请生成完整的代码。
```
