# 情绪天气 — 24节气动态壁纸网站设计文档

## 1. 项目概述

**项目名称**: 情绪天气  
**类型**: 24节气动态壁纸 / 文化展示网站  
**目标平台**: Web（桌面端 + 移动端）  
**核心定位**: 通过传统水墨美学与现代交互技术，呈现24节气的诗意意境，打造Hackathon演示级视觉体验。

**关键成功指标**:
- 首屏加载时间 < 3s
- 动画流畅度 60fps
- 支持键盘/触摸/鼠标三种交互方式
- 符合 WCAG 2.1 AA 无障碍标准

---

## 2. 风格方向

**设计系统**: UI/UX-Pro-Max 浅色水墨风（方案B）

| 元素 | 值 | 说明 |
|------|-----|------|
| 背景色 | `#f8f6f2` | 宣纸质感，温暖柔和 |
| 主文字色 | `#2a2a2a` | 浓墨色，对比度充足 |
| 次要文字色 | `#6a6a6a` | 淡墨，用于诗句与描述 |
| 辅助色 | `#8a8a86` | 节气名称、装饰元素 |
| 强调色 | `#d5d0ca` | 远山、云雾层叠 |
| 交互高亮 | `#2a2a2a` | 当前圆点、hover状态 |

**字体选择**:
- 节气名称: `Songti SC`, `SimSun`, serif — 宋体，传统感
- 诗句: `Kaiti SC`, serif — 楷体，书法感
- 描述文字: `Noto Serif SC`, serif — 清晰易读
- 字号比例: 节气名 12px → 诗句 36px → 描述 13px，形成清晰层级

```

|  +------------+  |
|  |  底部圆点   |  |  <- 水平居中，指示当前页
```

### 3.2 页面层级（z-index）

| 层级 | 内容 | z-index |
|------|------|---------|

### 3.3 内容对齐

- 垂直居中：flex + `align-items: center; justify-content: center`

- 描述卡片：absolute 定位，底部 `80px`，水平居中，max-width `320px`

## 4. 动画设计

### 4.1 四层水墨动效

| 层级 | 元素 | 实现方式 | 动画参数 |
|------|------|----------|----------|
| 1 | 远山剪影 | SVG `<path>` | 静态 + 半透明层叠，营造纵深感 |
| 2 | 墨迹晕染 | SVG `<ellipse>` + `feGaussianBlur` | `cx` 位移 10s, `opacity` 呼吸 10s |
| 3 | 飘落墨点 | SVG `<circle>` | `cy` 下落 12-18s, `opacity` 淡入淡出 |
| 4 | 底部飘雾 | SVG `<path>` + `animate` | `d` 路径形变 8s, 循环流动 |

### 4.2 页面切换动画

- **触发**: 点击箭头 / 键盘 / 圆点 / 触摸滑动
- **效果**: 当前页淡出 + 新页淡入，配合内容轻微位移（`translateX`）
- **时长**: 350ms
- **缓动**: `cubic-bezier(0.4, 0, 0.2, 1)` — 自然缓入缓出
- **性能**: 仅使用 `transform` 和 `opacity`，避免触发重排

### 4.3 入场动画

- 首屏加载：背景层先显示（无动画），文字内容延迟 300ms 淡入（`opacity 0→1`，`translateY 20px→0`）
- 描述卡片：延迟 600ms 从下方滑入

### 4.4 Reduced Motion

- 检测 `prefers-reduced-motion: reduce`
- 关闭所有连续动画（墨迹、墨点、飘雾）
- 保留页面切换动画，但时长缩短至 150ms

## 5. 交互设计

### 5.1 导航方式

| 输入方式 | 操作 | 行为 |
|----------|------|------|
| 鼠标 | 点击左箭头 | 切换到上一节气 |
| 鼠标 | 点击右箭头 | 切换到下一节气 |
| 鼠标 | 点击底部圆点 | 跳转到对应节气 |
| 键盘 | `←` 左箭头键 | 切换到上一节气 |
| 键盘 | `→` 右箭头键 | 切换到下一节气 |
| 键盘 | `Esc` | 跳转回第一个节气（立春） |
| 触摸 | 左滑 | 切换到下一节气 |
| 触摸 | 右滑 | 切换到上一节气 |

### 5.2 触摸优化

- 箭头点击区域：`44px × 44px`（符合 Apple HIG / Material Design）
- 圆点点击区域：`24px × 24px`（视觉上 6px，热区扩大）
- 元素间距：`≥8px`，防止误触
- 滑动阈值：水平位移 ≥50px 触发切换
- 垂直滚动：不拦截，允许页面正常滚动（如内容超出视口）

### 5.3 Focus 状态

- 箭头获得焦点时：显示 `2px solid #2a2a2a`  outline
- 圆点获得焦点时：放大至 `8px`，显示 outline
- Tab 顺序：左箭头 → 右箭头 → 圆点组

### 5.4 Hover 状态

- 箭头 hover：`border-color` 加深，`opacity` 从 0.6 → 1
- 圆点 hover：非当前圆点放大至 `8px`
- 过渡时长：`200ms`

## 6. 内容结构

### 6.1 数据模型（单条节气）

```typescript
interface SolarTerm {
  id: number;           // 1-24
  name: string;         // 如 "立春"
  poem: string;         // 如 "春风如贵客"
  subtitle: string;     // 如 "一夜东风起，万物竞生辉"
  description: string;  // 如 "立春是二十四节气之首，标志着冬季结束、春季开始，万物复苏。"
}
```

### 6.2 24节气列表

立春、雨水、惊蛰、春分、清明、谷雨、立夏、小满、芒种、夏至、小暑、大暑、立秋、处暑、白露、秋分、寒露、霜降、立冬、小雪、大雪、冬至、小寒、大寒

### 6.3 每页展示内容

1. **节气名称**：顶部，小字号（12px），宽字间距（12px），淡色（#8a8a86）
2. **诗句**：中心，大字号（36px），中等字间距（8px），主色（#2a2a2a）
3. **副标题**：诗句下方，中等字号（14px），斜体，淡色（#6a6a6a）
4. **描述卡片**：底部，白底半透明（rgba(255,255,255,0.9)），圆角 4px，阴影，12-13px 正文

## 8. 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 语义化结构 |
| Tailwind CSS v4 | Utility-first CSS 框架，负责布局、响应式、间距、颜色系统 |
| 自定义 CSS | 水墨动画关键帧、字体定义、复杂 SVG 样式 |
| SVG | 水墨背景动效 |
| Vanilla JavaScript | 轮播逻辑、事件监听、动画控制 |

### 8.1 Tailwind CSS v4 配置

**为什么选 Tailwind v4**：
- **零运行时**：编译为纯 CSS，无 JS 运行时开销，首屏加载快
- **JIT 引擎**：只生成用到的 class，CSS 体积极小（预计 < 10KB gzipped）
- **CSS 自定义属性**：v4 原生支持 `@theme`，与水墨风颜色系统无缝结合
- **响应式**：内置 `sm/md/lg/xl` 断点，与设计方案对齐

**自定义主题配置**：
```css
@theme {
  --color-paper: #f8f6f2;
  --color-ink-dark: #2a2a2a;
  --color-ink-mid: #6a6a6a;
  --color-ink-light: #8a8a86;
  --color-mountain: #d5d0ca;
  --font-song: 'Songti SC', 'SimSun', serif;
  --font-kai: 'Kaiti SC', serif;
  --font-serif: 'Noto Serif SC', serif;
}
```

**Tailwind 负责**：布局（flex/grid）、间距（p-*/m-*）、响应式（md:text-4xl）、颜色（text-ink-dark）、圆角、阴影
**自定义 CSS 负责**：SVG 动画关键帧、复杂渐变、字体 @font-face、特殊交互状态

### 8.2 性能策略

- 图片：无位图，纯 CSS + SVG，零图片请求
- 字体：使用系统字体栈（`Songti SC`, `Kaiti SC`, `Noto Serif SC`），无需加载外部字体
- 动画：CSS `@keyframes` + `transform`，GPU 加速
- 事件：节流处理 resize 和 touch 事件
- CSS 体积：Tailwind v4 JIT + 自定义主题，最终 CSS < 15KB gzipped

## 9. 无障碍支持

- **对比度**：主文字 `#2a2a2a` 对背景 `#f8f6f2` = 11.5:1，远超 AA 标准
- **键盘导航**：完整支持，可见 focus 状态
- **ARIA**：
  - 轮播容器：`role="region"`, `aria-label="24节气轮播"`
  - 箭头：`aria-label="上一个节气"` / `aria-label="下一个节气"`
  - 圆点：`role="tablist"`, 当前项 `aria-selected="true"`
- **Reduced Motion**：尊重系统设置，关闭连续动画
- **语义化标题**：每页节气名使用 `h2`，页面整体包含 `h1`（sr-only）
